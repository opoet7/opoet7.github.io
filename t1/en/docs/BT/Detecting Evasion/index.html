Detecting Evasion

یکی از دشوارترین مراحل برای تیم آبی شناسایی تکنیک هایی می باشد که نفوذگر با استفاده از آن از آنتی ویروس و یا EDR عبور کرده است در این مرحل نفوذ گر آشنایی کامل با محصولات امنیتی دارد و از Signature ها و Rule های آن ها آگاهی کافی دارد در این بخش قصد داریم تکنیک هایی را که گروه هایی APT در سال های گذشته از آن استفاده نموده اند را بررسی کنیم

Process Injection​

Process Injection is a versatile technique that adversaries leverage to perform a wide range of malicious activity. It’s so versatile that ATT&CK includes 14 sub-techniques of Process Injection

Event ID 25: ProcessTampering (Process image change)​

tamering.xml

<Sysmon schemaversion="4.50">
  <EventFiltering>
      <ProcessTampering onmatch="exclude">
      </ProcessTampering>
  </EventFiltering>
</Sysmon>
sysmon -i tampering.xml
event.module : "sysmon"  and event.code :"25"

Scenario : Detecting Procss Herpaderping​

event.module : "sysmon"  and event.code :"25"

DLL Injection With Signed Binary​

Adversaries may abuse mavinject.exe to proxy execution of malicious code. Mavinject.exe is the Microsoft Application Virtualization Injector, a Windows utility that can inject code into external processes as part of Microsoft Application Virtualization

C:\Windows\system32\mavinject.exe pid /INJECTRUNNING nss.dll
mavinject.exe ((Get-Process lsass).Id) /INJECTRUNNING C:\Windows\System32\vbscript.dll

Detection​

sysmon Event ID 10 Tracks ProcessAccess or cross-process events, generating a log entry whenever one process opens another process. Therefore, this event ID will likely generate a lot of noise without filtering out known benign cross-process behavior.

Event ID 4663: LSASS Access

Sysmon Event ID 7 The image loaded event logs when a module is loaded in a specific process

process where process.name == "powershell.exe"
and
handle_granted_to != null

Powershell Process Injection​

Windows API Call

  • VirtualAlloc / VirtualAllocEx / VirtualProtect

  • OpenProcess

  • WriteprocessMemory

  • CreateRemoteThread

  • ResumeThread

Powershell Event ID : 4104

  • PowerShell execution / display code execute in powershell .ps1

KQL

powershell.file.script_block_text : ("OpenProcess" or "VirtualAllocEx" or "WriteProcessMemory" or "CreateRemoteThread")

EQL 

event.category:process and powershell.file.script_block_text : (
(VirtualAlloc or VirtualAllocEx or VirtualProtect or  LoadLibraryA or 
or OpenProcess or OpenProcessToken or AdjustTokenPrivileges) and
(WriteProcessMemory or CreateRemoteThread 
CreateThread or ResumeThread))
